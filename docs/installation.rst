============
Installation
============


Stable release
--------------

To install tidy2tidy, run this command in your terminal:

.. code-block:: console

    $ pip install tidy2tidy

This is the preferred method to install tidy2tidy, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for tidy2tidy can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/mjkarlsen/tidy2tidy

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/mjkarlsen/tidy2tidy/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/mjkarlsen/tidy2tidy
.. _tarball: https://github.com/mjkarlsen/tidy2tidy/tarball/master
